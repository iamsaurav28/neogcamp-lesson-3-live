AUTOCAD ( drawing tools )

  1. LIMITS – It is set of limit in which u can change grid display and save it as currect model.

  2. UNITS – It is for control display ,u can change length,type, angle.

AUTOCAD (commands)

  1. LINE – with line, we can use a series of continous line segment. Each segment is a line object that can be edited separately.

  2. ERASE – erase ll help you to remove object which you dont want.

  3. CIRCLE – easily we can make circle ,shortcut key for it is "C" u can also make circle with radius,diameter etc.

  4. ARC – you can make it with centerpoint,endpoint,and also with start point.

  5. RECTANGLE - short-cut command for RECTANGLE is "Rec",u can also make rectangle with corners length width height.

  6. MIRROR – it is reflection of image , we can easily select image and mirror it.
