AUTOCAD ( drawing tools )

  1. LIMITS – It sets and control the limits of the grid display in the current model or named layout.the following prompts and displayed;

  2. UNITS – It controls the display and format of coordinate and angles as well as the orientation of angles.

ATUOCAD (commands)

  1. LINE – with line, we can use a series of continous line segment. Each segment is a line object that can be edited separately.

  2. ERASE – erase helps to remove the object froma drawing. to erase the unwanted objects

  3. CIRCLE – drawing a circle in autocad is rather simple. the default method for drawing a circle is to specify a center points and a radius. you can also choose to create a circle based on diameter, circumference, or area.

  4. ARC – to create an arc, you can specify combination of center,endpoint,start point,radius,angle,chord length and direction value. except the first method, arcs are drawn counterclockwise from the start point to the end point.

  5. RECTANGLE - With this command, you can speicfy the rectangle parameter (length,width,rotation) and control the type of corners (fillet, chamber or square).

  6. MIRROR – It creates a mirror image of the selected object. the user defines two points, along which autocad generates a "line of reflection" nd the reflected object is generated across this line with all components reversed.
